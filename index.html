SpriteAlive (prototype)

Java, Spring

Ideated from my fuse bead hobby, "choosing colors is the hardest part!!" I needed a way to go from video game sprites to fuse beads easily. I had just finished my intro programming course at my university, determined to create this tool, for myself, and others.


I was already keeping track of my beads in Google Sheets, so I learned about Google Apps Script to make an extension! It was pretty handy- the end result is at the end of this page.


This is really just a project for learning, since the codebase is pretty amateurish and only uses the Spring framework's embedded Tomcat server for deployment onto the Google Cloud platform. After I become more acclimated to standard web app practices at my university, I hope to do a full rewrite of this program later- as well as release it to the public, for others to use! (planning on Summer 2024)
